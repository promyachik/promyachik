Килиан Мбаппе

Килиан Мбаппе перешел из ПСЖ в Реал Мадрид - детали трансфера

ОФИЦИАЛЬНО 03.06.2024 Свободный агент

Источник: Официальное объявление Реал Мадрид

ТРАНСФЕРНАЯ НОВОСТЬ

Подробности перехода Килиана Мбаппе из ПСЖ в Реал Мадрид: статус сделки, контракт, статистика за предыдущий клуб и хронология трансфера.

Килиан Мбаппе стал игроком мадридского Реала после завершения контракта с Пари Сен-Жермен. Французский нападающий присоединился к испанскому клубу на правах свободного агента.

Переход завершил одну из самых продолжительных трансферных историй последних лет. Интерес мадридского клуба к футболисту обсуждался несколько сезонов, однако официальное объявление состоялось только после окончания его соглашения с ПСЖ.

Что известно о переходе Мбаппе в Реал Мадрид

Клуб подтвердил подписание нападающего 3 июня 2024 года. Футболист перешел без выплаты трансферной компенсации предыдущему клубу, поскольку его контракт с ПСЖ завершился.

Основные данные перехода:

  • предыдущий клуб - Пари Сен-Жермен;
  • новый клуб - Реал Мадрид;
  • тип перехода - свободный агент;
  • позиция - нападающий;
  • статус - официальный трансфер.

Результаты Мбаппе в ПСЖ

За время выступлений в Париже Мбаппе стал одним из главных игроков команды. Статистический блок на странице показывает основные показатели футболиста за предыдущий клуб.

Позже эти данные будут автоматически получаться через футбольный API. На странице можно будет отображать матчи, голы, результативные передачи и статистику по отдельным сезонам.

Почему этот трансфер важен

Переход усилил атакующую линию Реал Мадрид и объединил в одном составе нескольких ведущих футболистов своего поколения.

Для самого игрока трансфер стал новым этапом карьеры после нескольких сезонов во французском клубе.

Для ProFutbik эта страница станет центральным материалом по сделке. При появлении новой информации здесь можно обновлять условия контракта, статистику, связанные новости и ссылки на страницы игрока и клубов.

Что будет добавлено позже

После создания страниц игрока и клубов здесь появятся внутренние ссылки на:

  • профиль Килиана Мбаппе;
  • страницу Пари Сен-Жермен;
  • страницу Реал Мадрид;
  • историю всех переходов игрока;
  • последние новости;
  • последние матчи;
  • связанные трансферные материалы.

Страница будет обновляться по мере подключения новых данных и внутренних разделов сайта.
